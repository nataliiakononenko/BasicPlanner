# Basic Planner

A simple, intuitive Android calendar and planner app for managing your daily life.

## Features

- **Day View**: Outlook-style hourly timeline with events, drag to create
- **Week View**: Notebook-style layout with Mon-Wed on left, Thu-Sun on right
- **Month View**: Overview with event and todo indicators
- **Events/Meetings**: Create events with title, notes, start/end time
- **Recurring Events**: Daily, weekly, monthly, or yearly recurrence
- **To-Do Items**: Day-scoped or week-scoped tasks
- **Move to Next**: Tasks automatically move to next day/week until completed
- **Privacy First**: All data stored locally, no account needed

## Tech Stack

- Kotlin
- Android SDK 35
- SQLite for local storage
- Material Design components

## Building

Open the project in Android Studio and build using Gradle.

## Settings

- First day of week: Monday (default) or Sunday
